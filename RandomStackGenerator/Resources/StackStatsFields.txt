#строка, начинающаяся с решетки, игнорируется
Строка без пробелов, табов и знаков равенства. Идентификатор локации (ID). На карте названия локаций для размещения отрядов должны состоять из ID, знака подчеркивания и любого символа(ов). Примеры:$newline$В файле: MyLocId ; На карте (одна локация): MyLocId_A$newline$В файле: Тир2_1 ; На карте (две локации): Тир2_1_A Тир2_1_B$newline$В файле: Kraken ; На карте (три локации): Kraken_LegionLoc Kraken_ClanLoc Kraken_CommonLoc
ID
Целое число. Юниты с этой планкой опыта будут иметь больший шанс появиться в отряде. Для больших юнитов это значение умножается на $gm$
AverageExpBar
Целое число. Опыт за убийство отряда. То, на сколько точно генератор выполнит этот критерий, зависит от наличия подходящих юнитов
ExpStackKilled
Строка без пробелов, табов и знаков равенства. Список идентификаторов рас. Разделитель - плюс. Примеры:$newline$H$newline$L+U$newline$1+Water+K$newline$$newline$Идентификаторы рас:$newline$$ri$
Race
Целое число. Количество свободных слотов в отряде. Есть 10% шанс, что будет использовано на 1 слот меньше, и 10% - на 1 больше.
StackSize
Целое число. Максимальное количество юнитов, занимающих два слота. Если параметр равен нулю, но генератор не сможет поставить лидера, занимающего один слот, то он попытается поставить лидера, занимающего два слота
MaxGiants
Целое число. Генератор сначала постарается заполнить указанное количество слотов первого ряда
MeleeSlots
Целое число. Стоимость лута (предметы со стоимостью в золоте, равной нулю, не добавляются). При расчете стоимость драгоценностей делится на $jm$
LootCost
Строка. T, True, 1 - исключить из возможных сгенерированных предметов зелья, сферы, талисманы и свитки, Целое число - примерное количество (0 - случайное), Число - доля стоимости всего лута (0 - случайное)$newline$Пример:$newline$T#2#0.33$newline$$newline$
CItemsGen
Строка. T, True, 1 - исключить из возможных сгенерированных предметов надеваемые предметы и посохи, Целое число - примерное количество (0 - случайное), Число - доля стоимости всего лута (0 - случайное)$newline$Пример:$newline$T#2#0.33$newline$$newline$
NItemsGen
Строка. T, True, 1 - исключить из возможных сгенерированных предметов драгоценности, Целое число - примерное количество (0 - случайное), Число - доля стоимости всего лута (0 - случайное)$newline$Пример:$newline$T#2#0.33$newline$$newline$
JItemsGen
Строка без пробелов, табов и знаков равенства. Список идентификаторов содержимого лавки с предметами/заклинаниями/наемниками, либо параметра генерации (цена, тип или тип#цена для предмета, уровень для заклинания и планка опыта на 1 занимаемый слот для существа). Разделитель - плюс. Примеры:$newline$g000uu0001+200+700$newline$g000ss0005+g000ss0006+1RT+2CF (T - может быть глобальным, F - не может. H,L,C,U,E - раса, R - случайная раса)$newline$g000ig0002+1200+750+attack_artefact+sphere#300+6#400$newline$$newline$
ShopContent
Логическое. T, True, 1 - является внутренней охраной города
