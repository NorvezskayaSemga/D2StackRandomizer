genMode			genMode=1 Будет использован блок Common_map_settings и два блока Location - для стартовых и всех остальных локаций, соответственно/ngenMode=2 Будет использован блок Common_map_settings и все блоки Location - для каждой локации индивидуальные настройки

ApplySymmetry	Применять ли операции симметрии

SymmetryClass	-1 - случайная симметрия, если .ApplySymmetry=True, игнорируется, если .ApplySymmetry=False./nДля двух рас: 0 - L2, 1 - xM, 2 - yM, 3 - xy1M, 4 - xy2M./nДля четырех рас: 0 - L4, 1 - xM+yM, 2 - xy1M+xy2M./nL2 - поворот на 180 градусов вокруг оси, находящейся в центре карты/nL4 - поворот на 90 градусов вокруг оси, находящейся в центре карты. Только для квадратных карт/nxM - отражение в плоскости, параллельной оси Ox, и проходящей через центр карты/nyM - отражение в плоскости, параллельной оси Oy, и проходящей через центр карты/nxy1M - отражение в плоскости, параллельной диагонали {x, y}, и проходящей через центр карты. Только для квадратных карт/nxy2M - отражение в плоскости, параллельной диагонали {-x, y}, и проходящей через центр карты. Только для квадратных карт

xSize		Правая граница карты (например, если генерируем карту 24x48, то сюда пишем 24)
ySize		Верхняя граница карты (например, если генерируем карту 24x48, то сюда пишем 48)

RaceLocsDistTolerance			Генератор будет располагать локации со столицами так, чтобы для каждой из локаций выполнялось следующиее условие:/nR1*(1+RaceLocsDistTolerance) >= R2, при этом R2 > R1, где R1 и R2 - расстояние до двух ближайших локаций со столицами

PassageCreationChance			Определяет свободу перемещения по карте./n0 - генератор не старается соединять локации,/n1 - генератор соединит каждую локацию с каждой соседней (но стартовые старается не соединять в любом случае)

minPassWidth					Минимальная ширина проходов между локациями

AddGuardsBetweenLocations		Расставлять ли стражей проходов между локациями
PassGuardsPowerMultiplicator	Множитель силы стражей проходов между локациями
ObjectGuardsPowerMultiplicator	Множитель силы стражей посещаемых объектов

LocExpRatio				Отношение максимального опыта, получаемого за зачистку локации среднего размера, к минимальному./nЧем дальше локация от ближайшей столицы и чем ближе к центру, тем больше опыта за ее зачистку

StackStrength			Множитель силы нейтралов

Wealth					Множитель стоимости лута нейтралов

WaterAmount				Количество воды на карте. 0 - без воды, 1 - очень много
RoadsAmount				Количество дорог на карте. 0 - без дорог, 1 - максимальное количество
ForestAmount			Количество леса на карте. 0 - без леса, 1 - максимальное количество

CapitalRepulsion		Настройка расстояния от столиц до центра карты. 0 - столицы будут иметь более высокий шанс появиться рядом с центром карты. 1 - столицы будут иметь более высокий шанс появиться дальше от центра карты
